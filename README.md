# 🎮 Hangman Game (Examen TP JS)

A classic Hangman word-guessing game built using vanilla JavaScript, HTML, and CSS. This project was developed as part of a JavaScript practical exam.

## 🚀 [Live Demo](https://yassineelkefi.github.io/Hangman-Game/)

---

## ✨ Features
- **Dynamic Keyboard:** Interactive UI to select letters.
- **Visual Progress:** The hangman drawing updates with every wrong guess.
- **Word Categories:** (Optional: Mention if you have different themes like Animals, Movies, etc.)
- **Responsive Design:** Playable on both desktop and mobile devices.

## 🛠️ Technologies Used
- **HTML5:** For the game structure.
- **CSS3:** For styling and animations.
- **JavaScript (ES6+):** For the game logic, DOM manipulation, and event handling.

## 🕹️ How to Play
1. The game selects a random secret word.
2. Click on the letters on the screen to make a guess.
3. If the letter is in the word, it will be revealed.
4. If the letter is incorrect, a part of the hangman will be drawn.
5. Win by guessing the word before the hangman is fully drawn!

## 📂 Installation
If you want to run this locally:
1. Clone the repo: `git clone https://github.com/YassineElkefi/Hangman-Game.git`
2. Open `index.html` in your browser.
